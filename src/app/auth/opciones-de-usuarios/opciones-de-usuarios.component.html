Especialista

Este tipo de perfil es un medico contratado por nuestra clinica, que puede poseer una o varias especialidades. Una vez se haya registrado este usuario recibira un email de confirmación de identidad y ademas tendra que ser validado por un administrador del sistema.

Paciente

Este tipo de perfil es de una persona particular, para que pueda sacar turnos medicos para atenderse en nuestra clinica. Una vez se haya registrado este usuario recibira un email de confirmación de identidad.
